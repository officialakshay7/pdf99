What is Protect PDF?

Password-protecting a PDF adds encryption so that only someone with the correct password can open or edit the document. There are two types: an open password (required to view the file) and a permissions password (required to edit, print or copy content). PDF encryption uses AES-128 or AES-256 and is widely supported by PDF viewers on all platforms.
