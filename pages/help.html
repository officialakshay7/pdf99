How to Use PDF99

Step by step instructions for using PDF99 tools like Convert, Merge, Split and Compress securely online.

Step by Step Guide

Follow these three simple steps to process your files quickly and safely.

1. Choose a tool
Select the task (e.g. "PDF Converter" or "Merge PDF").
2. Upload or drag files
Drag files into the tool area or click to select from your device. You can select multiple files for batch actions where supported.
3. Configure & process
For tools with options (page ranges, quality, output format) choose settings, then start processing.
4. Download the result
When processing finishes, click download to save the file. For multi-file jobs you can download a ZIP or individual files.
5. Verify output
Open the downloaded file and check formatting. For critical documents, compare text, images and page order.
6. Troubleshooting tips
If something looks off: try different conversion settings, split the source file, or export a PDF from the original app first.

Quick Troubleshooting

Fast answers if you run into issues.
Upload fails / times out
Check network speed, try a different browser, or split the file if it's very large.
Output formatting differs
Complex layouts or embedded fonts can change. Export PDF from the source app or try alternate conversion settings.
Privacy concerns
Most tools run locally. When server-side processing is used, files are removed after a short window. See Privacy page for details.

Privacy & Security

We prioritize privacy and security. Key points:
  • Local-first processing: Many tools run entirely in your browser so files never leave your device.
  • Temporary server storage: When server processing is required we store files only for processing and remove them automatically within a short timeframe.
  • HTTPS everywhere: All connections use HTTPS/TLS.
  • No account required: For basic usage you can process files without creating an account.
  • GDPR friendly: We provide options to delete and manage files; contact support for removal requests.
