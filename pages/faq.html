Frequently asked questions

Short answers to common questions about usage, privacy and support.
Is my file uploaded to your servers?
By default most processing runs locally in your browser. When a server conversion is needed we only keep files temporarily for processing and remove them automatically. See Privacy & Security for details.
Are my files deleted after processing?
Yes - server-stored files are deleted after a short expiration window (typically minutes to a few hours). We do not keep processed files longer than necessary.
Which formats can PDF Converter handle?
Common office formats (DOCX, XLSX, PPTX), images (JPG, PNG, TIFF) and PDF â†” conversions. Some complex layouts may require manual checking.
Do I need an account to use the tools?
No - basic conversions and tools are available without creating an account. Accounts may be offered for advanced features like longer file retention, batch processing or history.
What about file size limits?
Each tool may have its own file size limits. If you encounter limits, try splitting large documents or using the Compress PDF tool before uploading.
My conversion looks different - why?
Complex elements (embedded fonts, macros, or advanced formatting) can cause differences. Check the output and manually adjust if needed; for critical documents consider exporting from the source application to PDF first.
How secure are the files during transfer?
All connections use HTTPS/TLS. When the service performs server-side processing, files are sent over encrypted channels and removed after processing.
Can I process multiple files at once?
Many tools support multiple file selection and batch actions (for example Merge PDF or JPG to PDF). Check the specific tool UI for batch capabilities.
Why can't I preview some files?
Preview depends on browser capabilities and file type. Large or unusual files may not render in-browser; in that case use download and open locally or try a different browser.
Where can I report a bug or request a feature?
Use the Contact or Support page to report issues or request features. Please include steps to reproduce and a sample file when possible.
Do you keep logs or analytics?
We may log minimal, non-sensitive analytics for performance and error monitoring. We do not store file contents in analytics logs.

Privacy & Security

We prioritize privacy and security. Key points:
  • Local-first processing: Many tools run entirely in your browser so files never leave your device.
  • Temporary server storage: When server processing is required we store files only for processing and remove them automatically within a short timeframe.
  • HTTPS everywhere: All connections use HTTPS/TLS.
  • No account required: For basic usage you can process files without creating an account.
  • GDPR friendly: We provide options to delete and manage files; contact support for removal requests.
