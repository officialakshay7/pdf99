What does rotating a PDF do?

Rotating changes the display orientation of pages in a PDF. It's a metadata operation — the actual page content is not re-rendered, so there is zero quality loss. Rotating is the correct fix for scanned documents where pages came out sideways or upside down, PDFs from mobile devices with incorrect orientation, or any document where the page layout doesn't match the intended reading direction.
